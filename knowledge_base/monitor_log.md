# 监控日志

> 由暗哨/虎哥维护。记录每个任务的执行过程和异常。日志保留7天，过期自动清理。
> 此文件仅钱守正/朱之文可查看。

## 日志格式

```
### 任务：<任务名称>
- **时间**: YYYY-MM-DD HH:MM
- **参与角色**: <角色列表>
- **执行过程**: 
  - HH:MM:SS - <角色> - <操作> - <耗时> - <结果>
- **异常标记**: [正常/异常] <描述>
- **合规检查**: [通过/不通过] <描述>
```

---

暂无监控记录。系统初建，首次任务执行后自动记录。

> 暗哨仅向钱守正/朱之文汇报异常。其他角色不可见此文件。
